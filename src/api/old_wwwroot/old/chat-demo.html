RAG Chat Widget Demo

How to use:

  1. First, create an assistant by making a POST request to /assistants
  2. Copy the returned assistant ID
  3. Replace YOUR_ASSISTANT_ID in the script tag below with your actual assistant ID
  4. The chat widget will appear in the bottom-right corner

Example Usage:

<!-- Add this script tag to any HTML page -->
<script src="https://localhost:5001/assistants/YOUR_ASSISTANT_ID/chat.js"></script>

Demo Content:

This is a demo page showing how the RAG chat widget can be embedded into any website. The chat widget will appear as a floating button in the bottom-right corner of the page.

Click the chat button to open the widget and start asking questions. The assistant will use the knowledge base you created when setting up the assistant.

Features:

  • Clean, modern UI with gradient styling
  • Floating toggle button
  • Real-time message display
  • Loading indicators
  • Error handling
  • Keyboard support (Enter to send)
  • Responsive design
  • Auto-scrolling messages
